# Описание задачи
Нужен скрипт который делает бекап всего проекта вместе с Docker images на флешке
## Сценарий использования
У нас есть docker-compose.yml который запускается на компьютере с интернетом. Нам нужно скопировать получившиеся образы, перенести их в закрытый контур и запустить на компьютере в закрытом контуре.

Допустим скрипт будет называться **docker_backup.sh**


## Как использовать скрипт:

1. **Сделайте скрипт исполняемым:**
```bash
chmod +x docker_backup.sh
```

2. **Запустите скрипт:**
```bash
./docker_backup.sh
```

## Особенности скрипта, например такие:

- **Автоматическое обнаружение** docker-compose.yml
- **Сохранение всех образов** из docker-compose.yml
- **Копирование контекста сборки** и init-скриптов
- **Интерактивный выбор** имени архива с датой/временем
- **Автообнаружение архивов** для восстановления
- **Создание скрипта восстановления** для закрытого контура
- **Подробное логирование** и цветной вывод
- **Проверка зависимостей** (docker, docker-compose, tar)

## Структура создаваемого архива может быть такой
```
resore_docker_имя_дата_время.tar.gz
├── docker-compose.yml
├── images/
│   ├── postgres-9.6-alpine.tar
│   ├── redis-alpine.tar
│   └── ...
├── init-db/ (если есть)
├── build_context/ (если есть)
├── restore.sh
└── README.md
```
